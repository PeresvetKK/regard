Заключение по результатам мониторинга

На основании запроса от 11.01.2022 об учащенном мониторинге здания окружающей застройки по адресу: СПб, Уткин пр., д. 13, к. 1, лит. А, в связи с превышением дополнительных деформаций, выполнен очередной цикл геодезического мониторинга.

По результатам выполненного геодезического мониторинга зданий окружающей застройки по адресу: Уткин пр. 13к1 лит А, по состоянию на 25 февраля 2022 года можно заключить следующее: 1. Приращение деформаций за текущий цикл не зафиксировано. 2. Максимальная суммарная дополнительная осадка зданий составляет 27 мм. 3. Фактические дополнительные осадки здания окружающей застройки (Уткин пр. 13к1 лит А) превышают допустимые (марка No46) (см. табл.1, прил. А, табл. А.1). 4. Относительная разность осадок здания (Уткин пр. 13к1) превышает допустимое значение (см. табл.1, прил. А, табл. А.2). 5. Дополнительный крен здания (Уткин пр. 13к1 лит А) превышает допустимое значение (максимальное значение превышения равно 0,0084>0,0020 на створе No2). 6. Раскрытие трещин за цикл не превышает 0,05 мм. 7. Максимальное раскрытие трещин составляет 0,55 мм (трещиномер Щ10), что не превышает предельно допустимого значения 3,0 мм (см. табл. 5.2.1. ГТО (шифр: 14228). ОАО «ЛЕННИИПРОЕКТ» КО, СПб, 2018).
